# Advanced Product Reviews for Magento 2

Premium Magento 2 (2.4.4–2.4.8) reviews suite by **eTechFlow**, in the style of
Amasty Advanced Product Reviews, plus three headline features:

- 🎥 **Video reviews** — customers upload and play back video with their reviews
- 🌐 **Auto-translation** — reviews translated on the fly via the Claude API
- 📊 **Analytics dashboard** — ratings, trends, and top-product insights in admin

Also includes: enhanced review form (pros/cons, recommend, photo upload),
verified-buyer badges, helpful votes, review comments, product Q&A,
review reminders with optional coupons, and a headless **GraphQL + REST** layer.

## Modules in this repository

| Folder | Module | Purpose |
| --- | --- | --- |
| [`AdvancedProductReviews/`](AdvancedProductReviews/) | `ETechFlow_AdvancedProductReviews` | Core module (Luma + headless API) |
| [`AdvancedProductReviewsHyva/`](AdvancedProductReviewsHyva/) | `ETechFlow_AdvancedProductReviewsHyva` | Hyvä theme compatibility (Tailwind + Alpine, GraphQL-driven) |

The Hyvä module is optional and only needed on Hyvä-themed storefronts.
See each module's own `README.md` for full installation and configuration details.

## Compatibility

- Magento Open Source (CE) & Adobe Commerce (EE/Cloud)
- Magento 2.4.4 – 2.4.8
- Themes: Luma, Hyvä, Hyvä Checkout
